# Exercise 1: Inventory Management System

## Scenario

We are developing an inventory management system for a warehouse where efficient data storage and retrieval are crucial.

## Objective

We aim to develop a system to manage products in inventory by performing the following operations:

* Add Product
* Update Product
* Delete Product
* Display Products

## Data Structure Used

HashMap<Integer, Product>

## Files

* Product.java
* InventoryManager.java
* Main.java

## Operations Implemented

1. Add Product
2. Update Product
3. Delete Product
4. Display Products

## Conclusion

We used HashMap to efficiently manage inventory records. The system supports fast insertion, updation, deletion, and retrieval of product information, making it suitable for handling large inventories.
