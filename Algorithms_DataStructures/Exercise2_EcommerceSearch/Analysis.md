# Time Complexity Analysis

### Linear Search

Linear Search checks each element one by one until the required product is found.

Time Complexity:

* Best Case: O(1)
* Average Case: O(n)
* Worst Case: O(n)

### Binary Search

Binary Search repeatedly divides the search space into two halves.

Time Complexity:

* Best Case: O(1)
* Average Case: O(log n)
* Worst Case: O(log n)

## Comparison

Linear Search is suitable for small or unsorted datasets. Binary Search performs better for large datasets but requires the data to be sorted.
