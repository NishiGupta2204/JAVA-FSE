# Exercise 2: E-commerce Platform Search Function

## Scenario

In this exercise, we worked on the search functionality of an e-commerce platform. The objective was to compare different searching techniques and understand their efficiency.

## Objective

The system performs the following operations:

* Store product information
* Search products using Linear Search
* Search products using Binary Search
* Compare the performance of both algorithms

## Algorithms Used

* Linear Search
* Binary Search

## Files Included

* Product.java
* SearchAlgorithms.java
* Main.java

## Operations Performed

* Search product using Linear Search
* Search product using Binary Search

## Conclusion

Linear Search is simple and works for unsorted data, whereas Binary Search is more efficient for sorted data. Binary Search reduces the search space and performs much faster for large datasets.
