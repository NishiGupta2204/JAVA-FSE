# Exercise 3: Sorting Customer Orders

## Scenario

In this exercise, we developed a system to sort customer orders based on their total price. Sorting orders helps in identifying high-value orders and improves order management.

## Objective

The system performs the following operations:

- Store customer orders
- Sort orders using Bubble Sort
- Sort orders using Quick Sort
- Compare the performance of both sorting techniques

## Algorithms Used

- Bubble Sort
- Quick Sort

## Files Included

- Order.java
- BubbleSort.java
- QuickSort.java
- Main.java

## Operations Performed

- Sorting orders using Bubble Sort
- Sorting orders using Quick Sort

## Conclusion

Bubble Sort is easy to understand and implement but is inefficient for large datasets. Quick Sort provides better performance and is generally preferred for sorting large collections of data.
