# Time Complexity Analysis

### Add Employee

Adding an employee takes O(1) time if space is available.

### Search Employee

Searching an employee takes O(n) time because we may need to traverse the array.

### Display Employees

Displaying employee records takes O(n) time.

### Delete Employee

Deleting an employee takes O(n) time because elements need to be shifted after deletion.

## Limitations of Arrays

Arrays have a fixed size and inserting or deleting elements in between requires shifting of elements. Therefore, arrays are suitable when the number of records is limited and known beforehand.
