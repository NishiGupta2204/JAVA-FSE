# Exercise 4: Employee Management System

## Scenario

In this exercise, we developed an employee management system for a company. The objective was to manage employee records efficiently using arrays.

## Objective

The system performs the following operations:

* Add employee
* Search employee
* Display employee records
* Delete employee

## Data Structure Used

Array

## Files Included

* Employee.java
* EmployeeManagementSystem.java
* Main.java

## Operations Performed

* Adding employee records
* Searching employee records
* Displaying all employees
* Deleting employee records

## Conclusion

Arrays provide simple and efficient storage for a fixed number of records. They are easy to implement and suitable when the size of data is known in advance.
