# Time Complexity Analysis

### Add Task

Adding a task at the beginning takes O(1) time.

### Search Task

Searching a task requires traversing the linked list, so the time complexity is O(n).

### Display Tasks

Displaying all tasks takes O(n) time.

### Delete Task

Deleting a task takes O(n) time because the required node must first be located.

## Advantages of Linked List

Linked lists provide dynamic memory allocation and make insertion and deletion easier compared to arrays. They are useful when the size of data is not fixed.
