# Exercise 5: Task Management System

## Scenario

In this exercise, we developed a task management system to maintain and organize tasks efficiently. A singly linked list was used to store task records dynamically.

## Objective

The system performs the following operations:

* Add a task
* Search a task
* Display all tasks
* Delete a task

## Data Structure Used

Singly Linked List

## Files Included

* Task.java
* TaskManagementSystem.java
* Main.java

## Operations Performed

* Adding tasks
* Searching tasks
* Displaying all tasks
* Deleting tasks

## Conclusion

Using a linked list allows dynamic memory allocation and efficient insertion and deletion of tasks. It is suitable when the number of records changes frequently.
