# Time Complexity Analysis

### Linear Search

Linear Search checks each book one by one until the required book is found.

Time Complexity:

* Best Case: O(1)
* Average Case: O(n)
* Worst Case: O(n)

### Binary Search

Binary Search repeatedly divides the search space into two halves.

Time Complexity:

* Best Case: O(1)
* Average Case: O(log n)
* Worst Case: O(log n)

## Comparison

Linear Search is easy to implement and works on unsorted data. Binary Search is more efficient but requires the records to be sorted before searching.
