# Exercise 6: Library Management System

## Scenario

In this exercise, we developed a library management system to search books efficiently. Both Linear Search and Binary Search algorithms were implemented to compare their performance.

## Objective

The system performs the following operations:

* Store book records
* Search books using Linear Search
* Search books using Binary Search
* Compare both searching techniques

## Algorithms Used

* Linear Search
* Binary Search

## Files Included

* Book.java
* SearchLibrary.java
* Main.java

## Operations Performed

* Searching books using Linear Search
* Searching books using Binary Search

## Conclusion

Linear Search is simple and works on unsorted data, whereas Binary Search is faster and more efficient for sorted records. Binary Search is preferred when dealing with large datasets.
