# Time Complexity Analysis

### Recursive Approach

The recursive function calculates the future value by repeatedly multiplying the current value with the growth rate.

Time Complexity:

* O(n)

where n represents the number of years.

## Optimization

The recursive solution is simple and easy to understand. However, for a large number of recursive calls, an iterative approach can be more efficient as it avoids the overhead of maintaining the recursion stack.
