# Exercise 7: Financial Forecasting

## Scenario

In this exercise, we developed a financial forecasting system to predict future values based on past growth rates. A recursive approach was used to calculate future values.

## Objective

The system performs the following operations:

* Calculate future value using recursion
* Predict growth over a given number of years
* Analyze the efficiency of the recursive solution

## Concept Used

* Recursion

## Files Included

* FinancialForecast.java
* Main.java

## Operations Performed

* Calculating future values recursively
* Predicting financial growth

## Conclusion

Recursion provides a simple and elegant way to solve repetitive problems. It helps break down a problem into smaller subproblems and makes the implementation easier to understand.
